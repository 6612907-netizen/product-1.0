# Flin 前端上手指南

## 立即开始

### 1. 安装 Cursor
去 [cursor.com](https://cursor.com) 下载安装

### 2. 打开项目
```bash
cd ~/product-1.0
open .
```
然后用 Cursor 打开这个文件夹

### 3. GitHub 登录
```bash
gh auth login
```
选 HTTPS，登录你的 GitHub 账号
